# SmartEng

## Tài khoản truy cập

Dưới đây là danh sách các tài khoản và mật khẩu mặc định (từ database) để đăng nhập vào ứng dụng:

### 🎓 Dành cho Học viên (Learner)
- **Tài khoản:** `learner01` - **Mật khẩu:** `Pass@1234`
- **Tài khoản:** `learner02` - **Mật khẩu:** `Pass@5678`
- **Tài khoản:** `learner03` - **Mật khẩu:** `Pass@abcd`
- **Tài khoản:** `learner04` - **Mật khẩu:** `Pass@efgh`
- **Tài khoản:** `learner05` - **Mật khẩu:** `Pass@ijkl`
- **Tài khoản:** `learner06` - **Mật khẩu:** `Pass@mnop`
- **Tài khoản:** `learner07` - **Mật khẩu:** `Pass@qrst`
- **Tài khoản:** `learner08` - **Mật khẩu:** `Pass@uvwx`

### ⚙️ Dành cho Quản trị viên (Admin)
- **Tài khoản:** `admin01` - **Mật khẩu:** `Admin@2024`
- **Tài khoản:** `admin02` - **Mật khẩu:** `Admin@2025`
- Hoặc bạn có thể gõ `admin` vào ô Username (với bất kỳ mật khẩu nào) để bypass và vào quyền Admin.
